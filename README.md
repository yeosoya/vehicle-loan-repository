ICT602 lab assignment
Mobile Technology Assignment (20%) 

This assignment must be done individually. 
This assignment evaluates students on: 

INSTRUCTIONS: 
1. Develop a mobile application for estimating monthly vehicle loan payments.
   
2. The inputs expected for the application are: 
i. Vehicle Price (RM) 
ii. Down Payment (RM)
iii. Loan Period (Years) 
iv. Interest Rate (%) per year 

3. The application is expected to output: 
i. Loan Amount = Vehicle Price – Down Payment 
ii. Total Interest = Loan Amount × (Interest Rate / 100) × Loan Period 
iii. Total Payment = Loan Amount + Total Interest 
iv. Monthly Payment = Total Payment ÷ (Loan Period × 12)
